# Food Delivery App – UI/UX Design (Figma)

This project is a complete mobile UI/UX design for a modern **Food Delivery App**, created as part of the **CodeAlpha Internship**.  
The design showcases a clean, user-friendly interface with intuitive navigation, consistent spacing, and smooth interactions built using **Figma**.

---

## Features
- Onboarding Screens  
- Login / Signup  
- Home Screen (Category Cards, Featured & Popular Restaurants)  
- Restaurant Details Page  
- Add to Cart Functionality (prototype interactions)  
- Cart Screen (quantity controls)  
- Checkout Screen (delivery & payment summary)  
- Order Success Confirmation  
- Fully linked prototype with clear transitions

---

## Tools & Technologies
- **Figma** — UI design, Auto Layout, Prototype  
- **GitHub** — Project hosting & documentation  
- PNG exports for reference/screenshot assets
