# Food Delivery App – UI/UX Design

This project is a **Food Delivery App UI/UX Design** created as part of the **CodeAlpha Internship Program**.  
It showcases a modern, clean, and user-friendly interface for browsing meals, selecting categories, placing orders, and tracking delivery.

The design was created using **Figma**, following best practices in layout, accessibility, visual hierarchy, and intuitive navigation.

## 🎯 Project Purpose

The goal of this project is to demonstrate:
- UI/UX design skills  
- Understanding of auto-layout, constraints, spacing, and responsive design  
- Ability to prototype navigation between screens  
- Professional documentation and presentation  

---

## ✨ Features Included

- Onboarding Screens – Welcome & Intro flow  
- Home Screen – Featured meals, categories, search bar  
- Food Categories – Filter meals by type  
- Meal Details Page – Description, price, add to cart  
- Cart Page – List of items, quantity adjustment  
- Checkout Page – Payment summary and order button  
- Order Success Page – Confirmation screen  
- Interactive Buttons & Navigation between screens
